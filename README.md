📊 Sales Analytics Dashboard

A comprehensive Sales Analytics Dashboard designed to provide deep insights into business performance across multiple dimensions. This dashboard helps visualize and analyze key sales metrics in a clear and interactive way, enabling data-driven decision making.

It includes detailed analysis of:

📦 Sales performance across product categories
🗺️ State-wise/region-wise sales distribution
💳 Mode of payment breakdown
📈 Profit and loss tracking
👥 Customer insights including top customers and regular buyers
🏆 High-value customers contributing maximum profit
📊 Overall business trends and performance metrics

This project demonstrates how raw sales data can be transformed into meaningful insights using visualization and analytics techniques, making it useful for business understanding and decision-making.
